# qoder-test# TODO アプリケーション

HTML、CSS、JavaScriptで作成されたシンプルなTODOアプリケーションです。フロントエンドのみで動作し、localStorageを使用してデータを永続化します。

## 🌟 主な機能

- ✅ 新しいTODOアイテムの追加
- ✅ TODO項目の完了/未完了の切り替え
- ✅ 既存のTODO項目の編集
- ✅ TODO項目の削除
- ✅ フィルタリング機能（すべて/進行中/完了済み）
- ✅ 完了済みアイテムの一括削除
- ✅ localStorageによる永続的なデータ保存
- ✅ レスポンシブデザイン（モバイル/デスクトップ対応）
- ✅ アクセシビリティ対応
- ✅ 日本語UI

## 🚀 使用方法

### ローカル環境での実行

1. **ファイルをダウンロード**
   ```bash
   git clone <repository-url>
   cd qoder
   ```

2. **ローカルサーバーを起動**
   ```bash
   # Python 3を使用する場合
   python3 -m http.server 8000
   
   # または Node.js の http-server を使用する場合
   npx http-server -p 8000
   ```

3. **ブラウザでアクセス**
   ```
   http://localhost:8000
   ```

### 直接ファイルを開く場合

`index.html` をブラウザで直接開くこともできますが、一部の機能（localStorageなど）が制限される場合があります。

## 🔧 機能詳細

### TODOの管理
- **追加**: 入力フィールドにテキストを入力してEnterキーまたは「＋」ボタン
- **完了切り替え**: チェックボックスをクリック
- **編集**: 「編集」ボタンをクリック、またはテキストをダブルクリック
- **削除**: 「削除」ボタンをクリック

### フィルタリング
- **すべて**: 全てのTODOを表示
- **進行中**: 未完了のTODOのみ表示
- **完了済み**: 完了済みのTODOのみ表示

### キーボードショートカット
- **Enter**: 編集を保存
- **Escape**: 編集をキャンセル
- **Tab**: フォーカス移動

## 🔒 データ永続化

- **ストレージ**: localStorage API使用
- **キー**: `todoApp_todos`
- **形式**: JSON文字列
- **容量制限**: ブラウザの制限に依存（通常5-10MB）

## ♿ アクセシビリティ

- **セマンティックHTML**: 適切なHTML要素とARIA属性
- **キーボードナビゲーション**: 全機能がキーボードで操作可能
- **スクリーンリーダー対応**: 適切なラベルと説明

## 🐛 トラブルシューティング

### よくある問題

**Q: データが保存されない**
- A: ブラウザのプライベートモードを無効にしてください
- A: localStorageが利用可能か確認してください

**Q: スタイルが適用されない**
- A: ローカルサーバー経由でアクセスしてください
- A: ブラウザのキャッシュをクリアしてください

**Q: 一部の機能が動作しない**
- A: JavaScriptが有効になっているか確認してください
- A: サポートされているブラウザを使用してください
